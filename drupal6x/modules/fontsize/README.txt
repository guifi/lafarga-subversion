Drupal Install and Setup:
******************************************************************************

1. Copy the fontsizer directory into your modules directory
2. Activate it under Administer > Modules
3. Configure permissions under Administer > Access control
4. Setup your fontsizer config Administer > Settings > Fontsizer

For more details http://www.yaml-fuer-drupal.de/tutorials/modul-fontsize/.


Icons:
******************************************************************************

All icons are configurable. Simply put your icons into "modules/fontsize/img" 
and change the filenames in Administer > Settings > Font Size.
